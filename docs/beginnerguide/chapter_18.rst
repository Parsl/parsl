18. Further Help and Resources
==============================

Community Forums and Support Channels
-------------------------------------

The Parsl community is active and welcoming to newcomers. If you encounter any issues or have questions, you can seek help through the following channels:

- **Parsl Slack Workspace**: Join the Parsl Slack workspace to connect with other Parsl users and developers. You can ask questions, share your experiences, and get help from the community.
- **Parsl Google Group**: The Parsl Google Group is another forum where you can ask questions and get help from the community.
- **GitHub Issues**: If you encounter a bug or have a feature request, you can open an issue on the Parsl GitHub repository.

Additional Documentation and Tutorials
--------------------------------------

- **Parsl Documentation**: The official Parsl documentation provides comprehensive information on all aspects of Parsl, from installation and configuration to advanced usage and customization. You can find the documentation online at: https://parsl.readthedocs.io/en/latest/
  
- **Parsl Tutorial**: The Parsl tutorial is a great way to start with Parsl. It provides a step-by-step guide on how to write and run your first Parsl script. You can access the tutorial online at: https://parsl.readthedocs.io/en/latest/tutorial.html

- **Parsl Examples**: The Parsl GitHub repository contains a collection of example scripts and configuration files that demonstrate how to use Parsl for various tasks. You can find the examples at: https://github.com/Parsl/parsl/tree/master/examples
